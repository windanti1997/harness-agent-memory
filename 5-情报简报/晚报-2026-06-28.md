## ⚡ 机会雷达

**Fintech Engineering Handbook（593 pts / 180 评论）** — 今天质量最高的技术内容。Voytek Pitula 写的一本系统化手册，覆盖金钱表示精度、复式记账、事件溯源、幂等性、对账、审计轨迹等核心模式。不只讲理论，给了完整端到端案例（加密提现、卡充值、含返现的换汇）。做支付/交易/量化系统的人直接当参考文档用。PDF/网页版免费。

## 💡 用户痛点 / 有趣项目

- **Wayfinder Router（77 pts / 28 评论）** — 不用 LLM 调用的 LLM 路由。纯靠分析 prompt 结构和措辞（长度/标题/列表/代码/数学关键词）判断难度，微秒级决定是丢给本地小模型还是云端大模型。离线、确定、可校准。核心卖点：省大模型 API 费用，决策本身零成本。和 LiteLLM/OpenRouter 这类 provider 网关是互补关系（路由决定走哪 tier + 网关决定走哪家）。适合 Agent 场景的 prompt 成本优化。
