# 📈 周报 W26-2026 (6/22 - 6/28)

## 📈 本周市场信号

### 1. 🇨🇳 中国AI开源反击：Z.ai 发布 GLM-5.2，744B 参数 MIT 许可
Z.ai（原智谱AI）在华盛顿封禁 Anthropic Fable 5 同周开源了 GLM-5.2（744B参数，MIT许可），SWE-bench Pro 62.1 分超越 GPT-5.5（58.6）。1M token上下文窗口，面向长周期编程任务。API成本仅为 GPT-5.5 的 1/6，且无地域限制可自部署。同日 360 也推出了对标 Anthropic Mythos 的图灵蜂 AI。

**信号强度：⭐⭐⭐⭐** — 中美AI脱钩加速，开源模型能力首次在 coding benchmark 上实质性超越闭源对标产品。

### 2. 🌏 亚洲AI模型爆发：Sakana AI 发布 Fugu，日本版"神话"级模型
东京 Sakana AI 发布 Fugu 模型（李维斯·琼斯、David Ha 联合创立），自称"与 Anthropic Fable 5 和 Mythos Preview 并驾齐驱"。官网广告词：*"delivering frontier capability without the risk of export controls"*。专为 agentic workflow 优化，使用日本语料和文化数据，低成本小数据集训练。

**信号强度：⭐⭐⭐⭐** — 出口管制倒逼亚洲AI生态系统自主化，agentic 模型成差异化卖点。

### 3. 🤖 AI Agent 身份基础设施诞生周
本周三件大事：(1) 爱沙尼亚宣布全球首个 AI Agent 数字身份系统；(2) Linux Foundation 宣布启动 Agent Name Service（DNS式可移植身份基础设施）；(3) Martin Fowler 发长文《Building reliable agentic AI systems》（HN 195pts）。AI Agent 从"跑得通"进入"可信可靠可治理"阶段。

**信号强度：⭐⭐⭐⭐⭐** — 基础设施层的变化往往意味着赛道从早期探索进入规模化部署前夜。

### 4. 💸 Perp Futures 冲击华尔街，AI Agent 接入金融数据
WSJ 报道 CFTC 批准永续期货（Perp Futures）后引发交易所股票抛售和高调诉讼。同周 Show HN 出现 Drip——面向 AI Agent 的按次付费财经通讯/数据 API。AI Agent 直接消费金融数据产品的商业模式开始成形。

**信号强度：⭐⭐⭐** — 永续期货对传统交易所是威胁，但对 AI Agent 接入 DeFi 和结构化金融产品是增量机会。

### 5. 🛠️ AI Agent 工具生态持续繁荣
- **peerd**（74pts HN）：纯浏览器端 AI Agent 框架，零部署
- **AgentKits**：60个生产级 AI Agent 蓝图 + 护栏（HITL、信任等级等）
- **Ponytrail**：AI 编码 Agent 本地审计追踪
- **Polygraph**：跨仓库会话记忆
- **Fika Jobs**：获 $4M 种子轮，AI Agent 做面试官

**信号强度：⭐⭐⭐** — 工具碎片化严重但快速收敛，"代理治理"成为新关键词。

## 🎯 潜在机会

1. **AI Agent 身份/合规 SaaS** — 爱沙尼亚政府和 Linux Foundation 推 Agent 身份体系，面向 Agent 的 KYC/身份管理/审计追踪工具缺口大。参考 Ponytrail + Agent Name Service 方向，做一个"Agent 护照"管理平台。

2. **亚洲 AI 模型中间层** — Sakana Fugu、GLM-5.2 证明了非美模型可行。做这些模型的**中文/日文适配 Agent 框架**，利用低成本 API 和零地域限制打造面向亚洲市场的 Agent 产品（成本比 OpenAI 低 5-6 倍）。

3. **AI Agent 金融数据中间件** — Drip 的思路可放大：做 AI Agent 可消费的结构化金融数据 API，尤其是永续期货等新资产的实时数据 + 分析信号。量化 + AI Agent 交叉点。

4. **科研论文工具方向的 AI Agent** — 本周虽无明显新竞品，但 GLM-5.2 的 1M token 上下文 + 低成本 API 天然适合对长上下文有强需求的论文分析/写作场景。做一个针对中国科研用户端的 AI 论文助手（用 GLM-5.2 自部署，成本可控且无监管风险）。

5. **"被制裁受益者"咨询/部署服务** — 美国出口管制导致大量开发者无法使用 Anthropic 最新模型。帮助企业**迁移到 Z.ai/Sakana 等非美模型**的咨询+部署服务可能成为新刚需。

## ⚠️ 风险提示

- **地缘政治风险**：中美AI脱钩是本周最大变量。上周用 Anthropic Mythos/Fable 的产品可能面临 API 被切断的风险。依赖单一模型供应商的策略不可持续。
- **GLM-5.2 实用化差距**：benchmark 好看，但实际 coding 场景、中文文案生成等真实体验有待验证。vendor benchmark 注水常见，别盲目押注。
- **Agent 合规成本**：爱沙尼亚和 Linux Foundation 的 Agent 身份体系意味着 Agent 行业监管即将来临。先发优势重要，但合规成本也是护城河。
- **Perp Futures 监管不确定性**：CFTC 批准后引发诉讼，永续期货在美国的合法性和可扩展性仍有变数。
