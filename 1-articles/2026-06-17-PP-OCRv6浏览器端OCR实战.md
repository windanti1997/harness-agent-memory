# PP-OCRv6 浏览器端 OCR 实战 —— 给 Agent 装上"本地眼睛"

## 一句话总结
百度飞桨发布 PP-OCRv6，首次推出 Tiny(1.5M)/Small(7.7M)/Medium(34.5M) 三档模型，Tiny 在浏览器端单图仅需 **97ms**，支持 50+ 语言，精度大幅提升。AI Agent 可在本地浏览器完成 OCR，数据不上云。

## 核心要点

### 关键数据
- **Tiny (1.5M)**：浏览器端单图最低 **97ms**，纯前端低算力可用
- **Small (7.7M)**：嵌入式设备
- **Medium (34.5M)**：Intel Xeon CPU 端到端 1.40s（前代 5.2 倍速度）
- 文字检测精度提升 **4.9%**，识别精度提升 **5.1%**（vs PP-OCRv5）
- 单模型支持语言：4种 → **50种**
- 新增工业场景：电路板、数码管、CAD图纸、喷码点阵字符

### Agent 应用价值
- 用户数据无需上传云端，**隐私安全**
- Agent 可在浏览器端直接 OCR 处理图片/PDF
- 多语言支持覆盖全球场景
